React Component Lifecycle

Three main stages in a React component's life:

  • Mounting (once)
    When the component is rendered for the first time
  • Updating (n times)
    When the component updates its state
    or the parent component renders it with new props
  • Unmounting (once)
    When the parent components removes it
    (from a render() or because it is removed as well)
